template<NotReference T>
hyperion::Option class

Represents an optional value.

Template parameters
T - The type of the potentially-contained value

Contents

  • Reference
    • Public static functions
    • Constructors, destructors, conversion operators
    • Public functions

Every Option is either Some and contains a value, or None, and does not. Useful for things such as:

  • * Optional members/fields
  • * Members/fields that can be loaned or "taken"
  • * Optional function arguments
  • * Nullable types/pointers
  • * Return values for functions not defined over their entire input range

Note

While T can be a pointer, Option will NOT take ownership of any pointer it contains, and as such it is the responsibility of the user to call the destructor on a pointer contained in an Option. To have an Option take ownership of a pointer, use std::unique_ptr instead of a raw pointer.

T cannot be a reference. To make an Option store a reference, use std::reference_wrapper instead.
