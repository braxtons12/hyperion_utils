PosixDomain#

class PosixDomain#

PosixDomain is the StatusCodeDomain that covers status codes covering a platform’s specific implementation of errno values in addition to those strictly required by POSIX (those represented by Errno).

using PosixStatusCode = StatusCode<PosixDomain>#

using value_type = i64#

The value type of PosixDomain status codes is i64

inline constexpr auto as_generic_code(const PosixStatusCode &code) const noexcept -> GenericStatusCode#

Converts the given status code to a GenericStatusCode

This will convert the given code to its semantically equivalent counterpart in the GenericDomain.

Note

Not all status code values are convertible to the GenericDomain, even from domains fully compatible with GenericDomain and that satisfy ConvertibleToGenericStatusCode. In this case, they will map to Errno::Unknown. Codes of value Errno::Unknown will never compare as semantically equivalent.
