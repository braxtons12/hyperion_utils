Functions#

Note: all hyperion meta-functions are contained in namespace hyperion::mpl

group mpl_functions

Various meta-functions for determining the validity of type traits and meta-functions on all/any of the elements in an mpl::list

Functions

template<template<typename> typename ConditionType, typename RequirementType, typename List>
static inline constexpr auto any_type_satisfies(List list)#

Meta-programming function to determine that at least one Type in the mpl::list, List, satisfies that the value of ConditionType<Type> equals the value of RequirementType. For example: 

static_assert(
    any_type_satisfies<std::is_trivially_destructible,
                 std::true_type,
                 mpl::list<u8, u16, u32>>
    ::value);

Checks that std::is_trivially_destructible<T> is true for at least one of u8, u16, or u32

Template Parameters

  • ConditionType – The condition to check

  • RequirementType – The required value of the condition

  • List – The mpl::list of types to check the condition for

template<template<typename, typename...> typename ConditionType, typename RequirementType, typename CheckList, typename ArgList>
static inline constexpr auto any_type_satisfies_with_arg_list(CheckList _list, ArgList arglist) noexcept#

Meta-programming function to determine that, given the mpl::list CheckList, containing types CheckTypes..., and the mpl::list ArgumentList, containing types ArgumentTypes..., at least one type, Type, in the CheckList satisfies that the value of ConditionType<Type, ArgumentTypes...> equals the value of RequirementType, where RequirementType is a type that ConditionType will compared against. For example: 

static_assert(
    any_type_satisfies_with_arg_list<std::is_constructible,
                                     std::true_type,
                                     mpl::list<u64, umax>,
                                     mpl::list<u8>
    >::value);

will check that std::is_constructible<T, u8>::value is true for at least one of T equal u64 and/or umax

Template Parameters

  • ConditionType – The condition to check for each combination of <Evaluatee, Type>

  • RequirementType – The required value of ConditionType

  • CheckList – The mpl::list of each type to check ConditionType for

  • ArgList – The mpl::list of argument types to use as the additional arguments for ConditionType

template<template<typename> typename ConditionType, HasValue RequirementType, typename List>
static inline constexpr auto all_types_satisfy(List list) noexcept#

Meta-programming function to determine that every type Type in the mpl::list, List, satisfies that the value of ConditionType<Type> equals the value of RequirementType. For example: 

all_types_satisfy<std::is_trivially_destructible, std::true_type, mpl::list<u8, u16,
u32>>;

Checks that std::is_trivially_destructible<T> is true for each of T equal to u8, u16, and u32

Template Parameters

  • ConditionType – The condition to check

  • RequirementType – The required value of the condition

  • List – The mpl::list of types to check the condition for

template<template<typename, typename...> typename ConditionType, HasValue RequirementType, typename Evaluatee, typename Lists>
static inline constexpr auto all_lists_satisfy_for_type(Lists lists) noexcept#

Meta-programming function to determine that for every list of types, Types... in Lists, the value of ConditionType<Evaluatee, Types...> is equal to the value of RequirementType

For example: 

all_lists_satisfy_for_type<std::is_constructible, std::true_type, std::string,
    mpl::list<mpl::list<usize, char>, mpl::list<const char*, usize>>>;

will check that std::is_constructible<std::string, Types...>::value is true for Types... equal to the list usize, char and equal to the list const char*, usize.

Template Parameters

  • ConditionType – The condition to check for each combination of <Evaluatee, Types...>

  • RequirementType – The required value of ConditionType

  • Evaluatee – The type that ConditionType should compare or evaluate each type, Type, in List against

  • Lists – The mpl::list of each mpl::list of types to pass to ConditionType
