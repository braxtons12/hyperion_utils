template<typename Der, typename Base>
concept Derived#
#include “Hyperion/Concepts.h”

Concept requiring that Der is derived from Base

Der will be tested as the unqualified, pointer-removed underlying type if Der is not a raw type (it will be tested as std::remove_pointer_t<std::decay_t<Der>>). EG: if Der is const volatile T*, it will be tested as T

Base will be tested as the unqualified, pointer-removed underlying type if Base is not a raw type (eg, it will be tested as std::remove_pointer_t<std::decay_t<Base>>) EG: if Base is const volatile U*, it will be tested as U
